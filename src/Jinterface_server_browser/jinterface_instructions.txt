Installera: sudo apt-get install erlang-jinterface

Lägga till i eclipse: preferences->installed JREs->Add->OtpErlang.jar
sökvägen för OtpErlang kan hittas genom att köra code:root_dir(). från Eshell

Importera bibliotek: import com.ericsson.otp.erlang.*;

Skapa erlang node med namn "enode" och cookie "erlang": erl -sname enode -setcookie  erlang
